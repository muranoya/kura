iframe Login Form

A login form embedded inside a same-origin <iframe>. The autofill dropdown should appear inside the iframe when focusing input fields.

Expected Behavior
  • Form type: LOGIN (inside iframe)
  • Content Script is injected into iframe via all_frames: true
  • Focus username/password inside iframe → dropdown appears inside the iframe
  • Select entry → fields inside iframe filled correctly
  • Dropdown is constrained to iframe boundaries (cannot overflow outside)
