Shadow DOM Login

Login form inside an Open Shadow Root. Tests focus event bubbling across shadow boundary.

Expected Behavior
  • Form type: LOGIN
  • Open Shadow DOM内のinputがfocusイベントバブリングで検出される
  • Focus username field → dropdown appears (Shadow DOM input基準で位置決定)
  • Select entry → Shadow DOM内のfields filled correctly
  • Closed Shadow Root inputs are NOT detected (known limitation)
