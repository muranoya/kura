Autocomplete Attribute Variations

Each field has a specific autocomplete attribute. Verify correct FieldType classification.

Expected Behavior
  • This page tests field classification, not full form flow
  • autocomplete="username" → username field
  • autocomplete="current-password" → password field
  • autocomplete="new-password" → new_password field (no dropdown)
  • autocomplete="one-time-code" → TOTP field
  • autocomplete="cc-name/cc-number/cc-exp/cc-csc" → credit card fields
  • Each field should show dropdown when focused (if matching entries exist)

Login Fields

autocomplete="username"
autocomplete="current-password"
autocomplete="new-password"

OTP

autocomplete="one-time-code"

Credit Card Fields

autocomplete="cc-name"
autocomplete="cc-number"
autocomplete="cc-exp"
autocomplete="cc-csc"
