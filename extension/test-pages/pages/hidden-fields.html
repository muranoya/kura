Hidden Fields

Visible fields should be detected. Hidden fields should be excluded from autofill.

Expected Behavior

  • Only username and password (visible) fields detected
  • 5 hidden fields should all be IGNORED: display:none, visibility:hidden, opacity:0, zero-size, type="hidden"
  • Dropdown appears only on the two visible fields
