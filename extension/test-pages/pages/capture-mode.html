Manual Credential Capture Mode

Test the manual credential capture feature. Start capture mode from the extension popup's capture button (crosshair icon).

Expected Behavior
  • Click the capture button (crosshair icon) in the extension popup
  • Floating toolbar appears at top-right: "kura: フィールドを選択"
  • Hover over input fields → indigo highlight border appears
  • Click a field → popover with "ユーザー名" / "パスワード" / "スキップ" buttons
  • Assigned fields get badges showing their role
  • Toolbar shows summary of selected fields
  • "保存する" button saves credentials to vault → success toast
  • Escape key cancels capture mode
  • Only input fields are selectable (not buttons, divs, etc.)
